# E-sport Social Platforme Gaming 

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
composer install
cp .env.example .env
php -S localhost:8000 -t public
```

## Structure
```
├── frontend/          # React application
├── backend/           # PHP API
├── docs/              # Documentation
└── .github/           # GitHub templates & workflows
```
